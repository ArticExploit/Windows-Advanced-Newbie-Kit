Windows Advanced Newbie Kit, W.A.N.K for short, is a utility to run maintenance, troubleshoot and fix common windows issues.

## Usage
Either download the one of the files and run those or use this command on powershell
```
irm https://wank.artic.sh | iex
```
### Powershell Script
If you're going to run the powershell script by directly downloading it, consider running this command in PowerShell with Admin priviledges first, to allow for it to run properly.
```
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass -Force
```
